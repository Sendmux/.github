# Support

## Product and account support

For product help, account-specific questions, or private workspace details, email [contact@sendmux.ai](mailto:contact@sendmux.ai).

Do not put API keys, passwords, tokens, webhook secrets, customer data, private account details, or full production payloads in GitHub issues.

## Documentation and developer tooling

Use GitHub issues for public, reproducible problems with docs, SDKs, CLI, or MCP tooling.

Include:

- The affected repository, package, command, or docs page.
- What you expected to happen.
- What happened instead.
- A minimal reproduction using test data only.

## Security

Report suspected vulnerabilities through [SECURITY.md](https://github.com/Sendmux/.github/blob/main/SECURITY.md), not public issues.
